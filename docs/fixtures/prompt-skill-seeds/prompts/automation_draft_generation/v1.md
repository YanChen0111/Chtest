# Prompt: automation_draft_generation v1

## Agent

AutomationDraftAgent

## Purpose

Generate a reviewable pytest or Playwright AutomationDraft from a Requirement or
approved TestCase.

## Input Schema

```json
{
  "type": "object",
  "required": ["source", "target_framework", "project_language"],
  "properties": {
    "source": {"type": "object"},
    "target_framework": {"type": "string", "enum": ["pytest", "playwright"]},
    "project_language": {"type": "string"},
    "repository_context": {"type": "object"},
    "test_command": {"type": "object"},
    "context_artifact_ids": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["drafts", "used_context_artifact_ids"],
  "properties": {
    "drafts": {
      "type": "array",
      "items": {
        "type": "object",
        "required": [
          "target_framework",
          "title",
          "suggested_file_path",
          "draft_language",
          "draft_code",
          "execution_notes",
          "risk_notes"
        ]
      }
    },
    "used_context_artifact_ids": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Generate draft code for human review; do not claim it has been applied.
The draft must not write to business source files.
State fixture, selector, data, dependency, and environment assumptions in
`execution_notes` or `risk_notes`.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_GENERATE_AUTOMATION_DRAFT",
  "message": "The source case or repository context is insufficient.",
  "recoverable": true
}
```
