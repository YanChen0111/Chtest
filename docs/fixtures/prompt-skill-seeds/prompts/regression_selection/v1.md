# Prompt: regression_selection v1

## Agent

RegressionAgent

## Purpose

Select the smallest useful pytest regression set for a CICDRun based on changed
files, risk analysis, available TestCommand records, and recent evidence.

## Input Schema

```json
{
  "type": "object",
  "required": ["cicd_run", "changed_files", "risk_analysis", "test_commands"],
  "properties": {
    "cicd_run": {"type": "object"},
    "changed_files": {"type": "array"},
    "risk_analysis": {"type": "object"},
    "test_commands": {"type": "array"},
    "historical_test_results": {"type": "array"}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["recommended_test_command_ids", "reasons", "risk_coverage", "needs_review"],
  "properties": {
    "recommended_test_command_ids": {"type": "array"},
    "reasons": {"type": "array"},
    "risk_coverage": {"type": "array"},
    "needs_review": {"type": "boolean"}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Select only provided TestCommand ids. Every recommended command must include a
reason tied to a changed file, risk reason, or prior evidence. If no provided
command covers a high-risk change, return `needs_review=true` and explain the
coverage gap.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_SELECT_REGRESSION",
  "message": "No usable TestCommand or risk evidence was provided.",
  "recoverable": true
}
```
