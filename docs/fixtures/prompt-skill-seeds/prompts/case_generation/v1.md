# Prompt: case_generation v1

## Agent

CaseGenerationAgent

## Purpose

Generate structured candidate test cases from a requirement, requirement review,
risk items, and optional local ContextArtifacts.

## Input Schema

```json
{
  "type": "object",
  "required": ["requirement", "target_test_types", "context_artifact_ids"],
  "properties": {
    "requirement": {"type": "string"},
    "requirement_review": {"type": "object"},
    "risk_items": {"type": "array"},
    "target_test_types": {"type": "array", "items": {"type": "string"}},
    "context_artifact_ids": {"type": "array", "items": {"type": "string"}},
    "context_manifest": {"type": "array"}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["cases", "used_context_artifact_ids"],
  "properties": {
    "cases": {
      "type": "array",
      "items": {
        "type": "object",
        "required": [
          "title",
          "priority",
          "test_type",
          "steps",
          "expected_results",
          "requirement_refs",
          "risk_refs",
          "ai_reason"
        ]
      }
    },
    "used_context_artifact_ids": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Generate review candidates, not official TestCase records.
Every case must include concrete steps and expected results.
Cover positive, negative, boundary, and risk-driven scenarios when supported by
the input.
Do not create vague cases such as "verify the feature works".

## Failure Output

```json
{
  "error_code": "UNABLE_TO_GENERATE_CASES",
  "message": "The input is insufficient to generate useful candidate cases.",
  "recoverable": true
}
```
