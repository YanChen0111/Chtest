# Prompt: knowledge_feedback v1

## Agent

KnowledgeFeedbackAgent

## Purpose

Convert accepted cases, rejected cases, review comments, failures, and reports
into draft TestKnowledgeCard feedback candidates for human review.

## Input Schema

```json
{
  "type": "object",
  "required": ["feedback_sources"],
  "properties": {
    "feedback_sources": {"type": "array"},
    "knowledge_evidence": {"type": "array"},
    "existing_knowledge_cards": {"type": "array"},
    "execution_evidence_summary": {"type": "object"},
    "forbidden_claims": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["knowledge_feedback", "unsupported_claims", "used_knowledge_evidence_ids"],
  "properties": {
    "knowledge_feedback": {
      "type": "array",
      "items": {
        "type": "object",
        "required": [
          "feedback_type",
          "draft_knowledge_type",
          "source_entity_type",
          "source_entity_id",
          "source_quote_or_hash",
          "source_span",
          "recommendation",
          "confidence",
          "review_findings"
        ]
      }
    },
    "unsupported_claims": {"type": "array", "items": {"type": "string"}},
    "used_knowledge_evidence_ids": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Create draft feedback only. Convert accepted cases into positive examples,
rejected cases into negative examples, review comments into TestStrategyNote
candidates, failures into BugPattern candidates, and missing coverage into
CoverageGap candidates. Keep source traceability at section, artifact, case,
run, report, or normalized KnowledgeEvidence level.

Runtime boundary:

- Do not call external providers.
- Do not create vector indexes.
- Do not trigger MCP runtime.
- Do not promote generated cases.
- Do not mark feedback as prompt-eligible; human review must do that.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_CREATE_KNOWLEDGE_FEEDBACK",
  "message": "The supplied feedback sources do not contain enough evidence.",
  "recoverable": true,
  "knowledge_feedback": [],
  "unsupported_claims": [],
  "used_knowledge_evidence_ids": []
}
```
