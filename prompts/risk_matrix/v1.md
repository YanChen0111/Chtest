# Prompt: risk_matrix v1

## Agent

RequirementReviewAgent

## Purpose

Convert requirement review findings into a structured risk matrix for test
design.

## Input Schema

```json
{
  "type": "object",
  "required": ["requirement", "review"],
  "properties": {
    "requirement": {"type": "object"},
    "review": {"type": "object"},
    "context_manifest": {"type": "array"}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["risks", "test_design_notes"],
  "properties": {
    "risks": {"type": "array"},
    "test_design_notes": {"type": "array"}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Every risk must include severity, likelihood, affected area, evidence, and a
test design note. Do not invent external evidence.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_BUILD_RISK_MATRIX",
  "message": "Requirement review evidence is missing or incomplete.",
  "recoverable": true
}
```
