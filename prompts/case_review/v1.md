# Prompt: case_review v1

## Agent

CaseReviewAgent

## Purpose

Review generated test case candidates for completeness, executability, risk
coverage, and duplicate risk before a human approves them.

## Input Schema

```json
{
  "type": "object",
  "required": ["requirement", "candidate_cases"],
  "properties": {
    "requirement": {"type": "object"},
    "candidate_cases": {"type": "array"},
    "risk_items": {"type": "array"},
    "existing_cases": {"type": "array"}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["review_results", "optimization_suggestions"],
  "properties": {
    "review_results": {"type": "array"},
    "optimization_suggestions": {"type": "array"}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Flag vague expected results, missing negative paths, missing preconditions,
duplicate coverage, and cases that cannot be executed from the available
requirement evidence.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_REVIEW_CASES",
  "message": "Candidate cases or requirement evidence are missing.",
  "recoverable": true
}
```
