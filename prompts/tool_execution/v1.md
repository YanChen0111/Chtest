# Prompt: tool_execution v1

## Agent

ToolExecutionAgent

## Purpose

Prepare a safe, allowlisted tool execution plan for pytest or Playwright based
on TestCommand and workflow context.

## Input Schema

```json
{
  "type": "object",
  "required": ["tool_definition", "test_command", "reason"],
  "properties": {
    "tool_definition": {"type": "object"},
    "test_command": {"type": "object"},
    "reason": {"type": "string"},
    "runtime_artifact_ids": {"type": "array"}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["allowed", "approval_required", "execution_plan", "risk_notes"],
  "properties": {
    "allowed": {"type": "boolean"},
    "approval_required": {"type": "boolean"},
    "execution_plan": {"type": "object"},
    "risk_notes": {"type": "array"}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Only approve commands that match ToolDefinition and TestCommand allowlists.
Reject arbitrary shell, destructive commands, network use without explicit
permission, and working directories outside allowed repository roots.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_PLAN_TOOL_EXECUTION",
  "message": "The tool command is not allowlisted or lacks required context.",
  "recoverable": true
}
```
