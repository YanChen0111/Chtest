# Prompt: coverage_analysis v1

## Agent

CoverageAnalysisAgent

## Purpose

Compare requirements, risks, generated cases, existing cases, execution
evidence, TestKnowledgeCards, and KnowledgeEvidence to identify coverage gaps
and duplicate risk.

## Input Schema

```json
{
  "type": "object",
  "required": ["requirements", "risk_items"],
  "properties": {
    "requirements": {"type": "array"},
    "risk_items": {"type": "array"},
    "generated_cases": {"type": "array"},
    "existing_cases_summary": {"type": "array"},
    "execution_evidence_summary": {"type": "object"},
    "knowledge_evidence": {"type": "array"},
    "forbidden_claims": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Output Schema

```json
{
  "type": "object",
  "required": ["coverage_gaps", "duplicate_risks", "unsupported_claims", "used_knowledge_evidence_ids"],
  "properties": {
    "coverage_gaps": {"type": "array"},
    "duplicate_risks": {"type": "array"},
    "covered_requirement_ids": {"type": "array", "items": {"type": "string"}},
    "covered_risk_ids": {"type": "array", "items": {"type": "string"}},
    "unsupported_claims": {"type": "array", "items": {"type": "string"}},
    "used_knowledge_evidence_ids": {"type": "array", "items": {"type": "string"}}
  }
}
```

## Instructions

Return JSON only. Do not include markdown fences in the model output.
Use only supplied local evidence. Identify missing P0/P1 requirement, risk,
boundary, exception, security, data, and regression coverage. Use GraphRAG-style
relationship reasoning only as a mental model over provided inputs; do not run
graph extraction. Cite KnowledgeEvidence and TestKnowledgeCard ids when
coverage findings depend on them.

Runtime boundary:

- Do not call external providers.
- Do not create vector indexes.
- Do not trigger MCP runtime.
- Do not promote generated cases.
- Do not run graph extraction, create graph jobs, mutate TestCase records, or
  create runner/report behavior.

## Failure Output

```json
{
  "error_code": "UNABLE_TO_ANALYZE_COVERAGE",
  "message": "The supplied requirements, risks, and evidence are insufficient for coverage analysis.",
  "recoverable": true,
  "coverage_gaps": [],
  "duplicate_risks": [],
  "unsupported_claims": [],
  "used_knowledge_evidence_ids": []
}
```
