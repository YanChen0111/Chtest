# Chtest Frontend

This directory contains the Vue 3 + Vite frontend for Chtest.

## Local Dev

Install dependencies and run the dev server locally:

```bash
npm install
npm run dev
```

Default local URL:

```text
http://localhost:5173
```

## Docker Compose Dev

The frontend service is wired through `deploy/docker-compose.yml` and runs the
same Vite dev server command inside the container.
